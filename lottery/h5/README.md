# 🎱 双色球彩票管理 H5

移动端网页应用，随时随地管理您的彩票记录。

## 功能特性

- 📝 **记录管理** - 添加、编辑、删除彩票购买记录
- 📷 **拍照识别** - 拍照自动识别彩票号码
- 🤖 **智能预测** - 本地算法 + Qwen3-Max AI 推荐
- 🔄 **飞书同步** - 支持同步到飞书多维表格
- 📊 **统计分析** - 中奖次数、总奖金统计

## 快速开始

### 方式一：直接打开
在浏览器中打开 `index.html` 文件即可使用。

### 方式二：本地服务
```bash
# 安装依赖
npm install

# 启动服务
npm start
```

然后访问 http://localhost:3000

### 方式三：部署到服务器
将整个项目上传到任何静态服务器即可。

## 技术栈

- 纯 HTML/CSS/JavaScript
- LocalStorage 本地存储
- WebRTC 相机访问
- 响应式设计 (适配手机浏览器)

## 使用说明

1. **添加记录**：点击首页"添加记录"按钮，输入期号并选择号码
2. **拍照识别**：点击"拍照识别"，授权相机后拍摄彩票照片
3. **查看预测**：切换到"预测"页，选择本地算法或AI推荐
4. **飞书同步**：在设置页配置飞书链接，可导入/同步数据
